# Proyecto-final-libertycity-

Aplicación web construida con React + Vite y Tailwind CSS. Proyecto integrador para la creación de una plataforma de gestión/visualización de libros.

Instrucciones rápidas:
- Instalar dependencias: `npm install`
- Levantar desarrollo: `npm run dev`

Descripción: este repositorio contiene componentes reutilizables en `src/components`, páginas en `src/Pages` y el punto de entrada en `src/main.jsx`.
